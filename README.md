## How to Run Locally

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Dev Server**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your browser.

## How to Build

### Web (PWA)
```bash
npm run build
# The output will be in the /dist folder
```

### Android
1. Setup Capacitor:
   ```bash
   npx cap add android
   npx cap sync android
   ```
2. Open in Android Studio:
   ```bash
   npx cap open android
   ```
3. Or build via command line (creates debug APK):
   ```bash
   cd android && ./gradlew assembleDebug
   ```

### iOS (Mac Only)
1. Setup Capacitor:
   ```bash
   npx cap add ios
   npx cap sync ios
   ```
2. Open in Xcode:
   ```bash
   npx cap open ios
   ```

## Installing on iPhone (GitHub Actions Artifact)

If you don't have a Mac set up for development, you can use the IPA file generated by GitHub Actions.

1. **Download the Artifact:**
   - Go to the **Actions** tab in this GitHub repository.
   - Click on the latest workflow run.
   - Scroll down to **Artifacts** and download `ios-unsigned-ipa`.
   - Extract the zip file to get `App.ipa`.

2. **Install using SideLoadly (Windows/Mac):**
   - Download and install [SideLoadly](https://sideloadly.io/).
   - Connect your iPhone to your computer via USB.
   - Open SideLoadly and drag `App.ipa` into the window.
   - Enter your Apple ID in the "Apple account" field.
   - Click **Start**.
   - Enter your password/2FA if prompted (this is sent to Apple for signing).

3. **Trust the App:**
   - On your iPhone, go to **Settings** -> **General** -> **VPN & Device Management**.
   - Tap your email under "Developer App".
   - Tap **Trust**.

4. **Enable Developer Mode (iOS 16+ only):**
   - Go to **Settings** -> **Privacy & Security**.
   - Scroll to bottom -> **Developer Mode** -> **On**.
   - Restart device.
